## Install

```bash
$ npm i egg-router-plus --save
```

## Usage

```js
// {app_root}/config/plugin.js
exports.router-plus = {
  enable: true,
  package: 'egg-router-plus',
};
```

## Configuration

```js
// {app_root}/config/config.default.js
exports.router-plus = {
};
```
